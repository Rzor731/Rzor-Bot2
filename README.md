# Rzor BOT
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install ffmpeg
> pkg install imagemagick
> git clone https://github.com/Rzor731/Rzor-Bot2
> cd Arya-Bot
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```

---------

### FOR WINDOWS/VPS/RDP USER
```bash
> git clone https://github.com/Rzor731/Rzor-Bot2
> cd Arya-Bot
> npm install
```
###### Run
```bash
> node index.js
```
